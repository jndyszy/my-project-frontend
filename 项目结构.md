# 项目结构

```
my-app/
├── node_modules/           # 第三方依赖包
├── public/                 # 静态资源（不参与编译，如 favicon.ico）
├── src/                    # 源代码核心目录
│   ├── api/                # 后端接口定义 (axios/fetch 请求)
│   ├── assets/             # 静态资源编译目录 (图片、全局样式、字体)
│   ├── components/         # 公共 UI 组件 (Button, Modal, Input 等)
│   ├── hooks/              # 自定义 React Hooks (如 useAuth, useLocalStorage)
│   ├── layouts/            # 页面布局组件 (如 MainLayout, AdminLayout)
│   ├── pages/              # 页面级组件 (对应路由跳转的完整页面)
│   ├── router/             # 路由配置 (React Router 定义)
│   ├── store/              # 状态管理 (Redux, Zustand 或 Context API)
│   ├── types/              # TypeScript 类型/接口定义 (.d.ts)
│   ├── utils/              # 工具函数 (日期处理、格式校验等)
│   ├── App.tsx             # 根组件 (应用入口点)
│   ├── main.tsx            # 全局入口文件 (挂载 DOM)
│   └── vite-env.d.ts       # Vite 类型声明
├── .gitignore              # Git 忽略文件配置
├── index.html              # 浏览器入口 HTML
├── package.json            # 项目依赖与脚本配置
├── tsconfig.json           # TypeScript 配置文件
└── vite.config.ts          # Vite 配置文件
```
